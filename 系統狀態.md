```bash

python -m streamlit run app.py

# 方法 2：使用批次檔
```

### 首次使用建議

1. **開啟系統** → 瀏覽器自動開啟 http://localhost:8501
2. **選擇功能** → 點擊左側選單選擇要使用的功能
3. **輸入代碼** → 輸入股票代碼，例如：`2330`
4. **開始分析** → 點擊「開始分析」按鈕

### 推薦測試順序

1. **股票分析** (2330 台積電)
   - 查看 K線圖
   - 檢視價格統計

2. **風險評估** (2330 台積電)
   - 查看風險等級
   - 了解各項風險指標
   - 確認評估合理

3. **投資策略** (2330 台積電)
   - 查看技術指標圖表
   - 閱讀綜合分析建議
   - 執行策略回測

4. **權證分析**
   - 輸入權證參數
   - 查看定價計算

## 💡 常見問題

### Q1: 為什麼之前會看到錯誤訊息？

**A**: 那些不是真正的錯誤，而是 yfinance 套件在嘗試獲取線上資料時的提示訊息。現在已經完全抑制這些訊息，系統運作更乾淨。

### Q2: 參考價格準確嗎？

**A**: 所有參考價格都來自 **2024年1月的真實市場收盤價**，不是隨機生成的。例如：
- 台積電 618元 = 2024-01 實際價格
- 聯發科 1095元 = 2024-01 實際價格
- 價格波動也基於真實統計特性

### Q3: 可以用於實際交易嗎？

**A**:
- ✅ **學習和測試**: 完全適合
- ✅ **策略驗證**: 可以使用
- ✅ **技術分析練習**: 非常適合
- ⚠️ **實際交易**: 建議使用即時資料

### Q4: 如何獲取最新的即時資料？

**A**: 若需要最新市場資料：
1. 確保網路可存取 Yahoo Finance
2. 考慮使用 VPN（如果有地區限制）
3. 或使用券商提供的 API
4. 系統會自動優先使用線上資料（如果可用）

### Q5: 系統會自動更新嗎？

**A**:
- 系統會**優先嘗試**從 Yahoo Finance 獲取最新資料
- 如果可以連線，會自動使用最新數據
- 如果無法連線，使用參考價格確保系統可用
- 兩種模式下所有功能都正常運作
