【專案概述】
─────────────────────────────────────────────────────────────
這是一個功能完整的台灣股市投資分析系統，整合了資料獲取、風險評估、
投資策略分析和權證評價等功能，提供投資者全方位的決策輔助工具。


【核心功能】
─────────────────────────────────────────────────────────────

1. 📊 股票分析
   ✓ 即時股價查詢
   ✓ 歷史價格走勢
   ✓ K線圖表顯示
   ✓ 基本面資訊
   ✓ 價格統計分析

2. ⚠️ 風險評估
   ✓ 波動率分析（年化）
   ✓ VaR 風險值（歷史模擬法）
   ✓ CVaR 條件風險值
   ✓ Beta 係數（系統性風險）
   ✓ Sharpe Ratio（風險調整報酬）
   ✓ 最大回撤分析
   ✓ 綜合風險評級（低/中/高風險）

3. 💡 投資策略
   ✓ 移動平均線（MA5/10/20/60）
   ✓ RSI 相對強弱指標
   ✓ MACD 指標
   ✓ KDJ 隨機指標
   ✓ 布林通道
   ✓ 綜合技術分析
   ✓ 策略回測功能

4. 🎯 權證分析
   ✓ Black-Scholes 定價模型
   ✓ Greeks 計算（Delta/Gamma/Theta/Vega/Rho）
   ✓ 內含價值與時間價值
   ✓ 理論槓桿與實質槓桿
   ✓ 損益兩平點計算
   ✓ 權證篩選與推薦
   ✓ 綜合評分系統


【技術架構】
─────────────────────────────────────────────────────────────

後端：Python 3.8+
  - 資料來源：twstock, yfinance
  - 資料處理：pandas, numpy
  - 技術分析：ta (Technical Analysis Library)
  - 統計模型：scipy, scikit-learn
  - 權證定價：Black-Scholes 模型

前端：Streamlit
  - 圖表：Plotly（互動式圖表）
  - 設計：響應式 UI
  - 介面：中文化完整支援


【快速開始】
─────────────────────────────────────────────────────────────

步驟 1：檢查環境
  python check_installation.py

步驟 2：安裝套件
  pip install -r requirements.txt

步驟 3：測試後端
  python test_backend.py

步驟 4：啟動系統
  streamlit run app.py
  或
  start.bat (Windows)
  ./start.sh (Linux/macOS)

步驟 5：開始使用
  瀏覽器開啟：http://localhost:8501

【使用範例】
─────────────────────────────────────────────────────────────

股票分析：
  1. 輸入股票代碼：2330（台積電）
  2. 選擇天數：90 天
  3. 點擊「開始分析」
  4. 查看 K線圖和統計資料

風險評估：
  1. 輸入股票代碼：2330
  2. 選擇天數：180 天
  3. 點擊「開始評估」
  4. 查看風險等級和各項指標

投資策略：
  1. 輸入股票代碼：2330
  2. 選擇天數：120 天
  3. 點擊「開始分析」
  4. 查看技術指標和操作建議
  5. 執行策略回測

權證分析：
  1. 輸入權證參數
  2. 點擊「分析權證」
  3. 查看理論價格和 Greeks
  4. 參考投資建議

【注意事項】
─────────────────────────────────────────────────────────────

1. 資料來源
   - 即時股價可能有延遲
   - 市場休市時無法獲取即時資料
   - 權證資料為示範性質

2. 風險聲明
   - 所有分析僅供參考，不構成投資建議
   - 投資有風險，請謹慎評估
   - 過去績效不代表未來表現

3. 技術限制
   - 首次載入需要下載歷史資料
   - 建議查詢天數控制在 90-180 天
   - 複雜計算可能需要等待

【常見問題】
─────────────────────────────────────────────────────────────

Q: 無法獲取股票資料？
A: 請確認：
   1. 股票代碼正確（台股代碼，如2330）
   2. 網路連線正常
   3. 非市場休市時間

Q: 安裝套件失敗？
A: 請嘗試：
   1. 更新 pip：pip install --upgrade pip
   2. 逐一安裝：pip install streamlit pandas ...
   3. 使用國內鏡像源

Q: 圖表顯示異常？
A: 請嘗試：
   1. 重新整理頁面（F5）
   2. 清除瀏覽器快取
   3. 更換瀏覽器
