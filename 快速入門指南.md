## 🎯 系統啟動

### 1. 環境需求

```bash
Python 3.11+
Streamlit 1.31.0+
```

### 2. 安裝套件

```bash
pip install -r requirements.txt
```

### 3. 啟動系統

```bash
streamlit run app.py
```

或使用 Python:

```bash
python -m streamlit run app.py
```

### 4. 開啟瀏覽器

系統會自動開啟瀏覽器，預設網址：
```
http://localhost:8501
```

## ⏱️ 5分鐘快速體驗

### 體驗 1: 股票基本分析 (2分鐘)

**目標**: 查看台積電的股價走勢

1. 點擊左側選單「📊 股票分析」
2. 在搜尋框輸入「2330」（台積電代碼）
3. 選擇查詢週期「90天」
4. 點擊「🔍 查詢股票」按鈕
5. 查看結果：
   - K線圖
   - 基本資訊
   - 歷史價格表

**預期時間**: 30秒

---

### 體驗 2: 技術指標分析 (2分鐘)

**目標**: 使用技術指標判斷買賣時機

1. 點擊左側選單「📈 技術分析」
2. 輸入股票代碼「2330」
3. 選擇分析週期「90天」
4. 點擊「🚀 開始分析」
5. 查看：
   - K線圖與成交量
   - MA均線圖（黃金交叉/死亡交叉）
   - MACD圖（趨勢強弱）
   - RSI圖（超買超賣）
   - KDJ圖（隨機指標）
   - 布林通道（波動範圍）
6. 滾動到底部查看「🎯 交易訊號」
   - 綜合訊號：買入/持有/賣出
   - 各指標明細

**預期時間**: 1分鐘

**範例結果**:
```
🟢 綜合訊號：買入

訊號詳情:
- MA: 買入 - MA5 上穿 MA20 (黃金交叉)
- RSI: 買入 - RSI = 28.5 (超賣)
- MACD: 持有 - MACD 接近信號線
```

---

### 體驗 3: 多股比較 (1分鐘)

**目標**: 比較台積電、鴻海、聯發科的表現

2. 輸入「2330,2317,2454」

**範例結果**:
```
比較摘要:

2330  +12.5%  22.3%   0.56

### 體驗 4: 建立投資組合 (1分鐘)

**目標**: 建立一個簡單的投資組合

1. 點擊「💼 投資組合」
2. 切換到「➕ 新增持倉」標籤
3. 新增第一支股票：
   - 股票代碼：2330
   - 股數：1000
   - 成本價：600
   - 點擊「➕ 加入組合」
4. 新增第二支股票：
   - 股票代碼：2317
   - 股數：2000
   - 成本價：110
   - 點擊「➕ 加入組合」
5. 切換到「📊 組合概覽」查看：
   - 總投資成本
   - 當前市值
   - 總損益與報酬率
   - 持倉明細
   - 持倉分布圓餅圖
   - 損益排名

**預期時間**: 1分鐘

**範例結果**:
```
總投資成本: $820,000
當前市值: $920,000
總損益: +$100,000 (+12.2%)
持股檔數: 2

## ❓ 常見問題

### Q1: 系統無法啟動怎麼辦？

**A**: 請確認：
1. Python 版本 >= 3.11
2. 已安裝所有套件：`pip install -r requirements.txt`
3. 嘗試重新安裝 Streamlit：`pip install --upgrade streamlit`

### Q2: 顯示 "無法獲取股票資料" 錯誤？

**A**: 可能原因：
1. **股票代碼錯誤**: 確認輸入的是正確的4位數代碼（例如：2330）
2. **網路問題**: 檢查網路連線
3. **API限制**: Yahoo Finance 可能暫時限制存取，稍後再試
4. **升級到終極版**: 安裝 FinMind 解決 429 錯誤
   ```bash
   pip install FinMind
   ```

### Q3: 技術指標圖表無法顯示？

**A**:
1. 確認已成功獲取股價資料
2. 檢查分析週期是否足夠（建議至少30天）
3. 某些指標需要較長的歷史資料（例如 MA60 需要至少60天）

### Q4: 投資組合中的股價不是最新的？

**A**:
1. 系統每次開啟組合頁面時會自動更新
2. 手動刷新頁面（F5）強制更新
3. 檢查網路連線與資料來源

### Q5: 如何理解恐懼貪婪指數？

**A**:
- **0-25**: 極度恐懼 → 可能是買入機會（逆向操作）
- **25-45**: 恐懼 → 可考慮分批進場
- **45-55**: 中性 → 保持觀望
- **55-75**: 貪婪 → 考慮獲利了結
- **75-100**: 極度貪婪 → 注意風險，避免追高

### Q6: 交易訊號準確嗎？

**A**:
- 交易訊號僅供參考，不保證準確
- 建議結合多種分析方法
- 技術分析有其局限性
- 實際投資前請諮詢專業顧問

### Q7: 資料是即時的嗎？

**A**:
- 系統使用 Yahoo Finance API
- 資料通常有 15-20 分鐘延遲
- 不適合當沖交易使用
- 適合中長期投資分析

### Q8: 可以用真實資金投資嗎？

**A**:
- 本系統僅供**教育與研究用途**
- 不提供下單功能
- 不構成投資建議
- 實際投資請透過合法券商

### Q9: 如何匯出分析結果？

**A**:
- 目前支援螢幕截圖
- 可使用瀏覽器的列印功能 (Ctrl+P) 存成 PDF
- 未來版本將支援 Excel/CSV 匯出

### Q10: 系統支援哪些股票市場？

**A**:
- 目前主要支援**台灣股票市場**
- 股票代碼格式：4位數字（例如：2330）
- 部分功能可支援其他市場（需修改代碼格式）
