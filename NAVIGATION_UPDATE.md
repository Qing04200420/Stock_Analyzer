# 🚀 首頁導航功能更新

## 更新日期
2026-01-10

## 功能說明

### 新增功能：首頁快速導航

在首頁的核心功能卡片區域，現在每個功能卡片下方都添加了可點擊的按鈕，讓使用者可以快速進入對應的功能頁面。

---

## 📋 功能特性

### 1. 一鍵跳轉

**功能卡片**：
- 📊 股票分析 → 點擊按鈕 → 直接進入股票分析頁面
- ⚠️ 風險評估 → 點擊按鈕 → 直接進入風險評估頁面
- 💡 投資策略 → 點擊按鈕 → 直接進入投資策略頁面
- 🎯 權證分析 → 點擊按鈕 → 直接進入權證分析頁面

### 2. 同步側邊欄

當點擊首頁按鈕跳轉後：
- ✅ 側邊欄的選單自動同步更新
- ✅ 顯示當前所在頁面
- ✅ 可以繼續使用側邊欄導航

### 3. 視覺設計

**按鈕樣式**：
- 漸層背景（紫色漸變）
- 白色文字
- 懸停效果（向上移動 + 陰影增強）
- 點擊反饋（按下效果）

---

## 🔧 技術實作

### 使用 Session State

```python
# 初始化頁面狀態
if 'current_page' not in st.session_state:
    st.session_state.current_page = "🏠 首頁"

# 按鈕點擊事件
if st.button("📊 進入股票分析", key="nav_stock"):
    st.session_state.current_page = "📊 股票分析"
    st.rerun()
```

### 同步側邊欄選單

```python
page = st.radio(
    "選擇分析功能",
    ["🏠 首頁", "📊 股票分析", ...],
    index=menu_options.index(st.session_state.current_page)
)
st.session_state.current_page = page
```

### CSS 樣式

```css
.stButton>button {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: white;
    border: none;
    box-shadow: 0 4px 8px rgba(102, 126, 234, 0.3);
    transition: all 0.3s ease;
}

.stButton>button:hover {
    transform: translateY(-3px);
    box-shadow: 0 6px 16px rgba(102, 126, 234, 0.5);
}
```

## 📊 使用流程

### 方式 1: 首頁快速導航（新增）

```
首頁 → 查看功能卡片 → 點擊按鈕 → 直接進入功能頁面
      └─ 1 步操作 ─┘
```

### 方式 2: 側邊欄導航（原有）

```
任意頁面 → 側邊欄選單 → 選擇功能 → 進入功能頁面
```

**兩種方式互補**：
- 首頁：快速瀏覽所有功能並選擇
- 側邊欄：在任意頁面間快速切換
