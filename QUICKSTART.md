# 快速開始指南

## 5分鐘快速上手

### 步驟 1：安裝相依套件

開啟終端機或命令提示字元，執行：

```bash
pip install -r requirements.txt
```

等待安裝完成（約需 2-3 分鐘）。

### 步驟 2：啟動應用程式

#### Windows 使用者
雙擊執行 `start.bat` 或在終端機執行：
```bash
start.bat
```

#### macOS/Linux 使用者
在終端機執行：
```bash
chmod +x start.sh
./start.sh
```

或直接執行：
```bash
streamlit run app.py
```

### 步驟 3：開始使用

應用程式會自動在瀏覽器中開啟（預設網址：http://localhost:8501）

如果沒有自動開啟，請手動在瀏覽器輸入：`http://localhost:8501`

## 功能示範

### 1. 股票分析（最簡單的功能）

1. 點擊左側選單的「📊 股票分析」
2. 在輸入框中輸入股票代碼，例如：`2330`（台積電）
3. 點擊「開始分析」按鈕
4. 等待 5-10 秒，查看分析結果

**你會看到：**
- 公司基本資訊（名稱、產業、市值）
- K線圖表
- 價格統計（最高、最低、漲跌幅）
- 歷史資料表格

### 2. 風險評估

1. 點擊「⚠️ 風險評估」
2. 輸入股票代碼：`2330`
3. 選擇分析天數：`180`
4. 點擊「開始評估」

**你會看到：**
- 風險等級評估（低/中/高風險）
- 波動率分析
- VaR 風險值
- Beta 係數
- Sharpe Ratio
- 最大回撤
- 投資建議

### 3. 投資策略

1. 點擊「💡 投資策略」
2. 輸入股票代碼：`2330`
3. 選擇分析天數：`120`
4. 點擊「開始分析」

**你會看到：**
- 綜合評分和操作建議
- 技術指標圖表（MA、MACD、RSI）
- 四大指標分析結果
- 可執行策略回測

### 4. 權證分析

1. 點擊「🎯 權證分析」
2. 選擇「權證評估」分頁
3. 輸入以下資訊：
   - 標的股票代碼：`2330`
   - 當前股價：`600`
   - 履約價：`650`
   - 權證類型：`認購`
   - 行使比例：`0.5`
   - 到期日：選擇未來 3 個月的日期
4. 點擊「分析權證」

**你會看到：**
- 理論價格計算
- 內含價值與時間價值
- 槓桿分析
- Greeks 指標
- 綜合評分
- 投資建議

## 常見問題

### Q1: 安裝時出現錯誤怎麼辦？

**A**: 請確認：
1. Python 版本是 3.8 或以上：`python --version`
2. pip 是最新版本：`pip install --upgrade pip`
3. 網路連線正常

如果仍有問題，可以逐一安裝套件：
```bash
pip install streamlit pandas numpy yfinance twstock plotly scikit-learn
```

### Q2: 無法獲取股票資料

**A**: 可能原因：
1. 股票代碼錯誤（請確認是台股代碼，例如：2330）
2. 網路連線問題
3. 市場休市（週末或國定假日）
4. API 限制（請稍後再試）

### Q3: 應用程式啟動緩慢

**A**: 正常現象，首次啟動需要：
1. 載入所有 Python 套件
2. 初始化 Streamlit
3. 建立快取

後續啟動會快很多。

### Q4: 圖表顯示不正常

**A**: 請嘗試：
1. 重新整理瀏覽器頁面（F5）
2. 清除瀏覽器快取
3. 使用其他瀏覽器（建議：Chrome、Firefox、Edge）

### Q5: 權證資料是真實的嗎？

**A**: 權證列表資料為示範資料。權證分析功能（Black-Scholes、Greeks）是真實計算，但需要輸入真實的權證參數才能得到準確結果。

## 進階使用技巧

### 1. 比較不同股票

開啟多個瀏覽器分頁，每個分頁分析不同股票，方便比較。

### 2. 匯出分析結果

在圖表上：
- 滑鼠移到圖表右上角
- 點擊「📷」圖示下載圖表

在資料表上：
- 可以複製貼上到 Excel

### 3. 客製化分析參數

- 調整查詢天數：短期（30-60天）vs 長期（180-365天）
- 調整風險評估參數：測試不同的分析期間
- 調整權證篩選條件：找到最適合的權證

### 4. 策略回測

使用「投資策略」頁面的回測功能：
1. 分析完成後，點擊「執行回測」
2. 查看歷史績效
3. 調整策略參數（修改程式碼）
4. 重新回測比較結果
